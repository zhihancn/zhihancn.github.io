错误的计算机概念

  1. 学计算机不等于装机器配电脑

  2. 计算机不等于PC（X86）

  3. 操作系统不等于Windows

  4. 网络不等于因特网 / 互联网（Internet）

  5. 局域网不等于Ethernet

容易混淆的概念

  1. Communication（通信）

    • Telecommunication（电信）
    • data communication（数据通信）

  2. network/net/web (network 的结果为networking)

    • computer network
    • inter-network Vs.intra-network

  3. networking

  4. internetworking

    • internet

  5. Internet，互联网，因特网(专有名词)

internet 和 Internet 的区别

  • 以小写字母“i”开始的 internet（互连网）是一个通用名词，它泛指由多个计算机网络互连而成的网络。

  • 以大写字母“I”开始的的 Internet（互联网或因特网）则是一个专用名词，它指当前全球最大的、开放的、由众多网络相互连接而成的特定计算机网络，它采用TCP/IP协议族作为通信的规则，且其前身是美国的ARPANET。

互联网 互连网
特指遵循TCP/IP标准、利用路由器将各种
计算机网络互连起来而形成的、一个覆盖全球的、特定的互连网
使用TCP/IP 除TCP/IP外，还可以使用其他协议
专有名词 是一个通用名词

[!TIP]
任意把几个计算机网络互连起来（不管采用什么协议），并能够相互通信，这样构成的是一个互连网（internet），而不是互联网（Internet）

术语表(Glossary)

Internet: network of networks
Edge 边缘网络

  1. end system（端系统），host（主机），application（应用）
  2. access networks接入网络

+residential network 住宅网
+company/institutional network 公司网
+wireless network 无线网
3. physical media 物理介质
+twisted pair双绞线
+coaxial cable同轴线缆
+optical fiber 光纤
+radio无线电

Core（switched network）核心网络

  1. circuit switching电路交换

+FDM/TDM Frequency/Time-division multiplexing频分多路复用/时分多路复用
2. packet switching 报文交换
+statistical multiplexing 统计多路复用
+store and forward 存储转发
+queue排队
3. heirarchy（tiers）层次结构

几种常见设备区分

  1. 以十亿计链接计算机设备

    • host(主机)= end system(终端)
    • running network apps(在终端上跑app)

  2. 传输设备

    • fiber(光纤)、copper(铜线传输)、radio(无线传输)、satellite(卫星，星链计划)
    • 传输速率:bandwidth(带宽)

  3. 包交换(由多个数据块封装形成包经行数据交换)

    • router(路由)和switches(交换机)

  4. Internet:”网络中的网络”

    • 英特网数据服务提供商

  5. 协议控制传输

  6. 因特网标准

    • RFC(请求评议，包含了关于Internet的几乎所有重要的文字资料。)
    • IETF(是一个公开性质的大型民间国际团体.是一个机构)

[!IMPORTANT]
协议定义了格式(语法错误),时序的收发(死循环)，和动作的产生(答案错误)

协议的通俗讲法：

  1. 计算机发送链接请求
  2. 服务器发送回应请求
  3. 计算机发送url
  4. 服务器返回文档

无线传输方式

  1. 以路由器为代表
    • 仅以局域网，信号覆盖面积小
    • 名称:802.11b/g/n
    • 商品名称 : WiFi
  2. 以型号基站为代表
    • 基站为建设主体
    • 常见4G,5G,高清视频通话
