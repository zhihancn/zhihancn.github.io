001两数之和

薛某 2020-04-25 22:16:12

题目描述：给定一个整数数组nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。

示例：

给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]

思路1 ：暴力法

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
       int  i = 0;
       int j = 0;
       for(i = 0 ; i<nums.size()-1;i++)
       {
           for(j = i+1 ; j<nums.size();j++)
           if(nums[i]+nums[j]==target)
           return{i,j};
       }
    return {0};
    }

};

其中 括号内的两个数据类型，一个是vector的nums，一个是int的target
