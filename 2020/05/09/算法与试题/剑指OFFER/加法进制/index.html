加法进制

薛某 2020-05-09 21:23:32

题目要求：
题目描述
写一个函数，求两个整数之和，要求在函数体内不得使用+、-、*、/四则运算符号。

示例

13+11 = ？;
13 的二进制      1 1 0 1    -----a        13
11 的二进制      1 0 1 1    -----b        11  

 (a&b) <<1  ->   1 0 0 1 0     -----d         18
          a^b  ->     0 1 1    -----e         6

 (d&e) <<1  ->   0 0 1 0 0    ------f        4
          d^e  ->  1 0 1 0 0   -----g        20

 (f&g) <<1  ->   0 1 0 0 0    ------h        8
          f^g  ->  1 0 0 0    ------i        16

 (h&i) <<1  ->   0           ------h        0      ---- --------退出循环
          h^i  ->  1 1       ------i           24

记住：num1^num2 是当前相加的结果
（num1 & num2）<< 1 表示当前结果是否需要进位

class Solution {
public:
    int Add(int num1, int num2)
    {
        int app=0;
        int add=0;
        do{
            app=num1^num2;//异或代表两数相加但不进位，于是 不进位和=真正的和-进位值
            add=(num1&num2)<<1;//进位值
            num1=app;
            num2=add;
        }while(add!=0);//当进位制为0时退出循环
        return app;
    }
};
